# 뉴앙 커뮤니티 의미 기반 컬러 시스템

문서 상태: 적용 기준
기준 테마: V12의 연하고 부드러운 중립 배경

## 목표

모든 클릭 요소를 보라색으로 표시하지 않는다. 색은 클릭 가능 여부가 아니라 사용자가 지금 보고 있는 행동의 의미와 상태를 빠르게 구분하는 데만 사용한다.

## 기본 중립색

- 본문: `#20232A`
- 보조 문구와 비활성 아이콘: `#6D7280`
- 구분선: `#E6E7EB`
- 중립 배경: `#F3F3F5`
- 앱 캔버스: `#F6F5F2`

작은 보조 문구는 기존 `#737887`보다 대비가 높은 `#6D7280`을 사용한다.

## 네 가지 의미색

### 1. 브랜드·창작 — 보라

- Strong: `#6657D9`
- Soft: `#EFEDFF`
- Border: `#D8D2FF`

사용:

- NUANG 워드마크와 코드 정체성
- 현재 하단 탭과 추천/데칼코마니 탭 표시
- 일반 글쓰기 진입과 게시 완료 CTA
- 화면에서 가장 중요한 브랜드 행동 하나

사용하지 않음:

- 좋아요
- 투표 막대
- 댓글과 답변
- 공유와 저장
- 단순 필터 활성 상태

### 2. 관계 반응 — 코랄

- Strong: `#C94B61`
- Soft: `#FFF0F3`
- Border: `#F3CCD4`

사용:

- 선택된 하트 아이콘과 좋아요 수
- 읽지 않은 알림 점과 배지

코랄은 감정적인 반응을 나타내므로 오류와 삭제에는 사용하지 않는다. 위험 행동에는 별도의 시스템 빨강을 사용한다.

### 3. 대화·기여 — 틸

- Strong: `#306E60`
- Soft: `#E6F4EF`
- Border: `#C7E5DC`

사용:

- 열린 댓글과 답변 흐름
- `뉴앙에게 물어봐`, `나도 질문하기`
- 질문 대상 선택
- 질문·답변 입력 포커스와 업로드 CTA

사용 이유: 댓글, 답변, 질문은 모두 다른 사람과 관계를 만드는 기여 행동이다. 성공 계열 민트와도 자연스럽게 이어진다.

### 4. 놀이·선택 — 오커

- Strong: `#8B621F`
- Soft: `#FBF3E4`
- Border: `#EBD8B6`

사용:

- 오늘의 성향 놀이터
- 선택된 밸런스 게임 답
- 투표 결과 막대
- 투표 다시 하기와 기록 진입

사용 이유: 보라색 브랜드 영역과 명확히 구분되면서도 앱의 따뜻한 캔버스와 조화를 이룬다. 두 선택지에는 같은 색을 사용하고 막대 길이만 달리해 정답이나 우열을 암시하지 않는다.

## 중립 상태를 유지하는 기능

- 공유: 항상 중립색. 완료 안내만 대화·성공 색을 사용할 수 있다.
- 저장: 기본 중립색, 저장됨은 검은색 채움 아이콘으로 표현한다.
- 필터: 적용 전후 모두 중립 계열. 선택 카드에는 검은 테두리와 체크를 사용한다.
- 검색과 알림 아이콘: 중립색. 새 알림 점만 코랄을 사용한다.

## 컴포넌트 규칙

1. 한 카드에서 의미색은 현재 활성 상태 하나에만 강하게 나타낸다.
2. 배경 전체를 의미색으로 채우지 않고 soft 배경이나 1px 선을 사용한다.
3. 의미색끼리 그라데이션하거나 색 그림자를 만들지 않는다.
4. 색만으로 상태를 전달하지 않는다. 아이콘 채움, 테두리, 글자, `aria-pressed`를 함께 사용한다.
5. 작은 텍스트에는 soft 색이 아닌 strong 색을 사용한다.
6. 일반적인 클릭 가능 여부는 색이 아니라 위치, 여백, 터치 크기와 피드백으로 알린다.
7. 새 기능을 추가할 때 기존 네 의미 중 하나로 설명되지 않으면 색을 추가하지 않고 중립색을 우선한다.

## 현재 적용 범위

- 좋아요 활성 상태: 코랄
- 댓글·답변 열림과 입력: 틸
- 뉴앙에게 물어봐와 질문 업로드: 틸
- 성향 놀이터 선택·통계·재투표: 오커
- 저장 활성 상태: 잉크색 채움
- 공유와 필터: 중립색
- 하단 메뉴, 코드, 일반 글쓰기: 브랜드 보라
