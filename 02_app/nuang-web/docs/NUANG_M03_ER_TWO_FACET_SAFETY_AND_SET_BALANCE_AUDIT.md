## 0. 감사 결론

ER의 두 facet은 정의와 핵심 반응 수준에서 구분 가능하다.

```text
ER-IR 감정 동요
좌절·의견 차이·작은 문제에서 불편한 감정이 얼마나 빠르게 커지고 이어지는가?

ER-WD 걱정·주저
불확실성·평가·실패 가능성에서 잘못될 생각이 얼마나 반복되어 행동을 망설이게 하는가?
```

24문항의 수량·방향·후보 상태는 균형을 이룬다. 하지만 다른 영역보다 안전·상태·낙인 위험이 크므로 바로 점수화할 수 없다.

핵심 위험:

1. 최근 큰 스트레스·수면·건강·돌봄 부담이 평소 성향보다 응답을 더 크게 바꿀 수 있다.
2. C는 건강하고 강한 사람, Q는 불안정하고 약한 사람이라는 도덕·임상 서열로 읽힐 수 있다.
3. 감정 동요는 외부 공격 행동, 걱정·주저는 신중함·계획·낮은 자신감과 섞일 수 있다.
4. 우울·자해·자살·공황·외상 내용을 가벼운 성향 코드에 포함하면 선별도 성향 측정도 제대로 수행하지 못한다.
5. 이 후보들은 마음속 반응만 측정하므로 실제 말·행동을 측정하는 `Q → C` 과정 프로필을 만들 수 없다.

현재 권장안:

> **공개 코어는 ER-IR+ER-WD 2-facet을 유지하되 일상적인 중간 강도 반응만 측정한다. 최근 상태는 별도 비채점 문맥으로 분리하고, 우울·위기·자살 위험 선별은 검증된 도구와 전문 대응 경로가 준비되기 전까지 코어에 넣지 않는다. C/Q는 정신건강 등급이 아니며 직접 응답·최근 상태·Q→C 과정은 공개하지 않는다.**

현재 판정은 `PASS_FOR_EXPERT_AND_COGNITIVE_REVIEW_WITH_CLINICAL_AND_STATE_FIREWALL`이다. 빠른·정밀 최종 문항 선정과 C/Q 발급은 `NOT_READY`다.

## 3. 성향·최근 상태 분리

### 3.1 성향 문항

```text
지난 6개월 동안 비슷한 상황에서 반복된 평소 반응
```

### 3.2 비채점 최근 상태 문맥 후보

정밀 결과를 해석하기 직전에 한 번만 선택적으로 묻는다.

```text
최근 2주 동안은 평소와 비교해 어땠나요?

평소와 비슷했어요
걱정이나 감정 반응이 조금 더 커졌어요
걱정이나 감정 반응이 훨씬 더 커졌어요
답하지 않을게요
```

이 질문은 진단·위기 선별이 아니고 C/Q 점수를 가감하지 않는다.

| 상태 응답    | 결과 처리                                                              |
| ------------ | ---------------------------------------------------------------------- |
| 평소와 비슷  | 일반 불확실성 문구 유지                                                |
| 조금 더 커짐 | `최근 상황의 영향이 일부 있을 수 있어요` 문맥 표시 후보                |
| 훨씬 더 커짐 | `최근 상태가 평소 성향보다 크게 반영됐을 수 있어요`와 재검사 선택 제공 |
| 답하지 않음  | 불이익·점수 변화 없음                                                  |

금지 처리:

- 최근 상태를 Q 가중치로 더하지 않는다.
- 최근 상태를 공개 프로필·공유·피드·비교에 넣지 않는다.
- `평소보다 커짐`을 질환·위기 신호로 자동 추론하지 않는다.
- 도움 연결은 특정 점수나 상태에만 숨겨진 방식으로 두지 않고 누구나 접근할 수 있게 한다.

## 4. 임상·위기 선별 방화벽

NIMH의 ASQ 사례는 자살 위험 선별이 짧은 질문만의 문제가 아니라, 검증된 도구·적용 대상·양성 후 추가 평가·전문 인력·임상 경로를 함께 요구한다는 점을 보여준다. 뉴앙 코어는 이런 구조를 갖추지 않은 채 유사 질문을 임의로 만들지 않는다.

향후 별도 선별 기능을 검토하려면 최소 아래 게이트가 필요하다.

1. 사용 목적과 대상 연령·지역이 명확한 검증된 도구
2. 한국어판의 출처·사용 허가·타당성 확인
3. 정신건강 전문가와 법률·개인정보 검토
4. 선별 전 고지·명시적 동의·건너뛰기
5. 양성 응답 후 즉시 이어지는 현재 지역의 대응 경로
6. 긴급·비긴급 상태별 안내와 훈련된 담당자
7. 최소 수집·고보호 저장·접근 로그·삭제 정책
8. 오탐·미탐·중단·연락 실패·야간 운영 대응
9. 출시 전 모의훈련과 정기 최신성 점검

이 게이트 전에는 코어 문항·자유 서술·알고리즘으로 위험을 추정하지 않는다.

## 7. HIGH·LOW와 C/Q 의미 대칭성

### 7.1 직접 의미

| 방향     | 허용되는 의미                                                | 금지되는 의미                                      |
| -------- | ------------------------------------------------------------ | -------------------------------------------------- |
| HIGH → Q | 불편한 감정·걱정이 비교적 빠르게 커지거나 반복됨             | 불안정, 약함, 예민함, 정신질환, 위험 감지가 뛰어남 |
| LOW → C  | 같은 상황에서 감정·걱정의 변화와 행동 방해가 상대적으로 작음 | 멘탈이 강함, 건강함, 회복력이 좋음, 감정이 없음    |

### 7.2 구조적 비대칭

C는 침착·건강·능력으로, Q는 취약·불안정으로 평가될 가능성이 높다. 방향 수를 12:12로 맞춰도 이 인상 차이는 사라지지 않는다.

필수 검증:

- 어느 방향이 더 좋은 사람·건강한 사람처럼 보이는지 별도 평정
- Q 응답을 숨기고 싶은 정도와 수치심·방어 반응
- C 응답이 감정을 억누르거나 무관심하다는 뜻으로 읽히는지
- Q 결과 문구가 진단처럼 느껴지는지
- 캐릭터·색·모션이 한 방향을 우월·위험으로 만들지 않는지

검사 화면에는 HIGH/LOW·C/Q 방향을 보여주지 않고, C에 차분한 미소·Q에 울거나 떨리는 캐릭터 같은 보상·낙인 반응을 사용하지 않는다.

- 진단이나 상담을 제공하지 않는 문항 이해 연구임을 알린다.
- 원하지 않는 질문은 건너뛰고 언제든 중단할 수 있음을 알린다.
- 구체적인 외상·위기 경험을 설명하도록 요구하지 않는다.

인터뷰 후:

- 문항 때문에 불편이 커졌는지 확인한다.
- 연구자가 결과·진단·개인 상담을 제공하지 않는다.
- 자발적인 위험 공개가 있을 때의 보호·연락·중단 절차는 모집 전에 전문가와 승인해 둔다.

## 10. 전문가·인지 인터뷰 출력

### 전문가 blind mapping

- 첫 번째·두 번째 facet 또는 어느 facet에도 맞지 않음
- HIGH/LOW 방향
- 일상 성향 vs 현재 상태 vs 임상 증상
- 내부 감정 vs 외부 표현 행동
- 실제 위험·정보 부족·환경 제약 오염
- 사회적 바람직성·낙인·불필요한 불편
- 유지 / 수정 / 제외와 근거

### 사용자 인지 인터뷰

- 실제 떠올린 상황의 강도·위협·손실
- 질문을 자기 말로 다시 설명한 내용
- 속감정·외부 행동·계획 검토·반복 걱정 중 무엇으로 답했는지
- 지난 6개월의 평소 모습과 최근 2주의 차이
- C/Q 또는 정신건강 검사라고 예상했는지
- 더 건강하거나 좋은 사람처럼 보인 방향
- 불편도·수치심·답변 회피·판단 어려움 이유

## 11. 정량 파일럿 모형

인지 검토를 통과한 문항으로 다음을 비교한다.

1. `ER-M2-CORRELATED`: IR·WD 두 상관 요인
2. `ER-M2-HIERARCHICAL`: 두 facet 위 상위 ER
3. 단일 ER 요인 대안
4. HIGH/LOW 문구·부정 제한·반복 시간 표현의 방법 요인
5. 최근 상태 공변량 포함·제외 민감도 모형
6. IR의 activation과 carryover가 갈리는 탐색 모형
7. WD의 worry와 hesitation이 갈리는 탐색 모형

비교 근거:

- 탐색·확인 표본에서의 구조 재현
- IR·WD의 구분 가능성과 상위 ER 적합성
- 교차 적재·국소 의존성·문항정보
- facet 및 상위 점수의 omega·표준오차
- 최근 상태 통제 전후 점수 안정성
- 연령·성별·생활 제약·관계 맥락별 불변성·DIF
- 사회적 바람직성·임상 오해 평정과의 관계
- 결과 문구 공감도·낙인·공개 수용도

하나의 적합도나 alpha만으로 승인하지 않는다. 임계값·표본·제외 규칙은 데이터를 보기 전에 사전 등록한다.

## 12. 빠른·정밀 검사 영향

### 빠른 코어

- 기존 ER 빠른 4문항은 모두 Q 방향이므로 교체한다.
- IR·WD 양쪽과 C·Q 양방향의 근거를 포함한다.
- 적은 문항으로 세부 facet을 확정적으로 설명하지 않는다.
- 최근 상태가 평소와 다르다고 해도 점수를 임의 보정하지 않는다.

### 정밀 코어

- IR·WD를 각각 보여주고 분리 프로필을 평균으로 숨기지 않는다.
- facet 방향이 달라도 `두 반응이 다르게 나타나요`로 설명한다.
- 실제 중심 경계는 `C와 Q가 비슷해요`로 구분한다.
- 직접 응답·원점수·최근 상태는 공개하지 않는다.

### Q→C 과정 연구

현재 24문항은 마음속 감정과 걱정을 측정한다. 외부 말·행동 층위를 같은 상황에서 별도로 측정하지 않았으므로 `Q → C`를 계산하거나 발급하지 않는다. 과정 연구는 별도의 paired item·안전·가시성 검토가 필요하다.

## 16. 사용자 승인 기록

2026-07-18 KST 사용자 승인으로 다음 네 문장을 확정했다.

1. ER-IR·ER-WD 24문항을 전문가·인지 인터뷰 후보로 검토한다.
2. 공개 코어에는 일상적인 불편 정서만 포함하고 임상·위기 선별 내용을 넣지 않는다.
3. 최근 2주 상태는 선택적 비채점 문맥으로 분리하고 C/Q 점수·공개 정보에 사용하지 않는다.
4. 현재 후보로 Q→C 과정 프로필을 만들지 않는다.

이에 따라 `RBL-01D`를 닫고 `RBL-01E — SE·OE·RO·SM·ER 전체 seam·상황·HIGH/LOW·문구 효과 통합 감사`로 이동한다.
