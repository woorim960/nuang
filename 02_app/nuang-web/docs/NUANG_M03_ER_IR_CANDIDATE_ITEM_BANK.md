## 0. 이번 단계의 결론

ER-IR은 감정 표현이 큰지, 화를 밖으로 드러내는지, 감정을 잘 조절하는지를 평가하지 않는다.

```text
사소한 좌절·의견 차이·예상 밖 문제·수정 피드백
→ 짜증과 불편한 감정이 얼마나 빠르게 커지고
→ 작은 불편이 반복되거나 다음 일을 할 때까지 얼마나 이어지는가
```

첫 후보 은행은 6개 상황 버킷, 총 12문항으로 구성했다.

| 구분                   |  수 |
| ---------------------- | --: |
| 높은 ER-IR 방향 `HIGH` |   6 |
| 낮은 ER-IR 방향 `LOW`  |   6 |
| 주력 인지 인터뷰 후보  |   8 |
| 경계 확인용 탐색 후보  |   4 |

`HIGH`는 Q의 증거 후보, `LOW`는 C의 증거 후보다. 문항 하나가 C/Q 글자를 확정하지 않는다. HIGH는 정신적으로 약하거나 공격적이라는 뜻이 아니고, LOW는 감정 조절력·회복력·합리성이 좋다는 뜻이 아니다.

## 1. 측정 목표와 금지 해석

### 측정하려는 것

- 사소한 좌절에서 짜증이 커지는 속도
- 일상적 의견 차이에서 답답함이 커지는 정도
- 예상하지 못한 작은 문제에서 불편한 감정이 활성화되는 속도
- 고칠 점을 들었을 때 감정 크기가 변하는 정도
- 작은 불편이 반복될 때 짜증이 쌓이는 정도
- 감정이 올라온 뒤 다음 일까지 이어지는 정도

### 측정하지 않는 것

- 소리 지르기·비하·공격·강압 같은 외부 행동
- 감정을 숨기거나 말로 표현하는 방식
- 기쁨·설렘·감동을 포함한 모든 감정의 풍부함
- 공감 능력·감수성·위험 탐지 능력
- 감정 조절 기술·회복력·문제 해결 능력
- 실제 스트레스 환경의 심각도
- 외상 반응·공황·조증·불안·우울 장애
- 자해·자살·폭력 위험 선별

## 2. 공통 응답·안전 계약

```text
최근 6개월 동안 비슷한 상황이 생겼을 때의 평소 모습

1 거의 그렇지 않아요
2 드문 편이에요
3 반반이에요
4 자주 그래요
5 거의 항상 그래요

이 상황은 답하기 어려워요
```

- 폭력·이별·사고·재난·괴롭힘처럼 강도가 큰 사건은 떠올리지 않는다.
- 현재 진행 중인 큰 위기나 최근의 예외적인 스트레스로만 답한 경우 `state_interference`를 기록한다.
- 감정이 올라온 것과 그 감정을 타인에게 어떻게 행동했는지를 분리한다.
- 감정이 올라온 뒤 진정시키는 방법을 얼마나 잘 쓰는지는 점수화하지 않는다.
- 문항 응답만으로 정신건강 상태나 안전 위험의 유무를 추론하지 않는다.
- 같은 상황의 HIGH/LOW 후보는 한 사용자 form에 함께 제시하지 않는다.

## 6. HIGH·LOW와 C/Q의 비진단 계약

| 방향          | 후보가 직접 말할 수 있는 것                                         | 금지되는 단정                                             |
| ------------- | ------------------------------------------------------------------- | --------------------------------------------------------- |
| HIGH → Q 증거 | 좌절·갈등·작은 문제에서 불편한 감정이 비교적 빠르게 커지거나 이어짐 | 정신적으로 약함, 예민함, 공격적임, 불안·우울이 있음       |
| LOW → C 증거  | 같은 상황에서 불편한 감정이 비교적 천천히 변하거나 짧게 이어짐      | 회복력이 좋음, 감정조절을 잘함, 이성적임, 정신건강이 좋음 |

Q와 C는 진단 등급이나 정신건강 점수가 아니다. HIGH/LOW 어느 쪽도 다른 사람을 대하는 행동의 적절성을 보장하지 않는다.

## 8. 인지 인터뷰용 분할

각 분할은 HIGH 2개·LOW 2개이며 같은 상황의 반대 후보를 포함하지 않는다.

| 분할   | 후보            | 주력 / 탐색 |
| ------ | --------------- | ----------: |
| `IR-α` | C01·C04·C09·C12 |       2 / 2 |
| `IR-β` | C03·C06·C08·C11 |       3 / 1 |
| `IR-γ` | C02·C05·C07·C10 |       3 / 1 |

필수 질문:

1. 어떤 강도의 사건을 떠올렸고 실제 손실이나 위협은 어느 정도였는가?
2. 속으로 감정이 커진 것과 밖으로 표현한 행동을 구분했는가?
3. 최근 큰 스트레스·수면 부족·건강 변화가 평소 응답을 바꿨는가?
4. LOW가 `감정 조절을 잘하는 좋은 답`, HIGH가 `정신적으로 약한 답`처럼 보였는가?
5. 문항이 불편하거나 임상 검사처럼 느껴졌는가?
