다섯 코드 자리 모두 검증된 범위의 상세 정보를 제공하고, 과정 화살표는 실제 두 단계가 분리 측정된 영역에만 사용한다. 대표 G/A와 `G → A`는 동일한 RO-EC 관계 주의 방향을 사용한다. `Q → C`와 `K → M`은 후속 연구 후보이며 E/I·R/N은 현재 facet 지도를 우선한다. RO-RN 선택·존중 신호는 사용자 본인의 정밀 리포트에는 보여주되 별도 공유 코드가 아니며 공개·공유·비교에서 기본 제외한다. SM-RL 맡은 일 이행은 3-facet 비교용 탐색 후보이며, 독립성·추가 설명력·비낙인성이 확인되기 전에는 K/M이나 공개·공유·비교에 합산하지 않는다. ER은 일상적인 감정 동요와 걱정·주저만 다루며, 임상·위기 선별을 코어에 섞지 않고 최근 상태·직접 응답·Q→C 과정 정보를 공개하지 않는다.
