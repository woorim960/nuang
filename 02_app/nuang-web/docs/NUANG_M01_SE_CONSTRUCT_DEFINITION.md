## 0. 이번 단계의 결정 요약

`SE 사람 사이 에너지`는 관계를 잘 맺는 능력이나 사람을 좋아하는 정도가 아니다. 일상적인 대인 상황에서 나타나는 아래 두 경향을 함께 관찰하려는 상위 축 가설이다.

1. `SE-RE`: 사람과 함께할 때 활력이 얼마나 쉽게 올라오고 교류에 빠져드는가
2. `SE-AI`: 필요한 상황에서 의견·요청·선택지를 얼마나 먼저 밖으로 꺼내는가

두 세부 성향은 관련될 수 있지만 같은 것은 아니다. 사람들과 있는 시간을 즐겨도 주도권은 다른 사람에게 맡길 수 있고, 혼자 있는 시간으로 회복하는 사람도 필요할 때는 의견을 분명히 먼저 말할 수 있다.

따라서 다음 원칙을 적용한다.

- 한 문항은 `SE-RE` 또는 `SE-AI` 하나만 측정한다.
- 두 세부 점수를 먼저 계산한 뒤에만 SE 상위 점수를 검토한다.
- 두 세부 점수가 크게 갈리면 평균으로 차이를 숨기지 않고 고객에게 `두 반응이 다르게 나타나요`라고 설명한다.
- SE 점수로 사교성, 리더십, 관계 능력, 인기, 공감 능력을 추론하지 않는다.
- 현재 SE 12문항은 자동 승계하지 않고 모두 재작성·재검증 후보로 돌린다.
- 정밀 결과에서는 `함께할 때의 에너지`와 `먼저 표현하기`를 별도 세부 성향으로 보여준다.
- 현재 두 facet을 `E → I`처럼 이어 붙이지 않는다. 실제로 먼저 표현하지 않은 이유를 내향성으로 자동 추론하지 않는다.

## 1. 이론적 위치와 뉴앙의 해석

DeYoung, Quilty, Peterson의 Big Five Aspect Scales 연구는 외향성 영역 안에 `Enthusiasm`과 `Assertiveness`라는 두 관련 하위 측면이 존재할 수 있다는 경험적 선례를 제시한다. 뉴앙의 `SE-RE`와 `SE-AI`는 각각 이 두 측면에서 출발한 것으로 해석할 수 있다.

다만 이것은 뉴앙의 한국어 문항이나 2-facet 구조가 이미 타당하다는 뜻이 아니다. 다른 검증된 모형인 BFI-2는 외향성 아래를 `Sociability`, `Assertiveness`, `Energy Level`의 세 facet으로 구분한다. 뉴앙은 사람과 함께하려는 성향과 그때의 활력을 `SE-RE` 하나에 묶으려 하므로, 아래 두 가설을 한국어 사용자 데이터에서 따로 확인해야 한다.

- `SE-RE`의 참여·즐거움·활력 내용이 실제로 하나의 세부 성향을 이루는가
- `SE-RE`와 `SE-AI`가 구분되면서도 하나의 SE 상위 축을 이루는가

이 문서의 정의는 문항을 만들기 위한 가설 명세이며, 타당성의 최종 증거가 아니다.

## 2. 명칭 체계 제안

| ID | 현재 내부 명칭 | 권장 내부 명칭 | 고객에게 보이는 쉬운 명칭 | 결정 |
|---|---|---|---|---|
| SE | 사람 사이 에너지 | 사람 사이 에너지 | 사람 사이 에너지 | 유지 제안 |
| SE-RE | 관계 참여 | 교류 활력 | 함께할 때의 에너지 | 변경 제안 |
| SE-AI | 주도적 표현 | 주도적 표현 | 먼저 표현하기 | 유지 제안 |

`관계 참여`는 관계 형성, 모임 참석, 관계 유지 능력까지 포함하는 것처럼 들려 측정 범위보다 넓다. 내부 명칭은 측정 대상을 분명히 하기 위해 `교류 활력`, 고객 명칭은 일상적인 표현인 `함께할 때의 에너지`를 권장한다.

내부 명칭은 개발·분석·전문가 검토에 사용하고, 고객 명칭은 검사 안내와 리포트에 사용한다. 두 명칭은 같은 `facet_id`와 `construct_definition_version`에 연결한다.

## 3. SE 상위 축 정의

### 3.1 한 문장 정의

`SE 사람 사이 에너지`는 일상적인 대인 상황에서 사람과의 교류로 활력이 올라오는 정도와, 필요한 의견·요청·선택지를 먼저 표현하는 정도를 함께 나타내는 상위 성향 축이다.

### 3.2 측정하는 것

- 사람과 함께하는 동안 또는 그 뒤에 느끼는 상대적인 활력 변화
- 대화·공동 활동에 자연스럽게 관여하고 빠져드는 경향
- 공동 결정이나 조율이 필요할 때 먼저 제안하는 경향
- 의견, 요청, 다른 생각을 적절한 때 밖으로 표현하는 경향

### 3.3 측정하지 않는 것

- 친구 수, 모임 횟수, SNS 활동량
- 사람을 좋아하거나 싫어하는 정도
- 인기, 매력, 유머, 카리스마
- 대화 기술, 발표 능력, 설득력
- 리더십 능력이나 조직 성과
- 친절, 공감, 배려, 협조성
- 관계 만족도나 관계 유지 능력
- 수줍음, 사회불안, 우울 등 임상적 상태
- 직업·학교·가족에서 맡은 역할이나 권한

### 3.4 높은 방향과 낮은 방향의 의미

기존 구현은 `T`(높은 방향), `S`(낮은 방향)를 사용했지만 사용자 검토에서 글자만으로 방향을 이해하기 어렵다는 문제가 확인됐다. [Code Order & Core Axis Review v3](./NUANG_CODE_ORDER_AND_CORE_AXIS_REVIEW_V3.md)에 따라 첫 자리는 `E / I`, 고객 자리 순서는 `SE → OE → RO → SM → ER`로 확정했다. 후속 SM 감사로 J/P를 폐기하고 K/M을 승인했으며 현재 작업 범례는 `E/I · R/N · G/A · K/M · C/Q`다. 아래 표의 E/I는 우열이 아닌 측정 방향을 나타낸다.

| 방향 | 관찰 가능한 경향 | 절대 의미하지 않는 것 |
|---|---|---|
| 높은 방향·E 후보 | 사람과의 교류에서 활력이 비교적 쉽게 올라오고, 필요할 때 의견이나 선택지를 먼저 꺼내는 경향 | 더 사교적이다, 더 좋은 리더다, 관계를 더 잘한다 |
| 낮은 방향·I 후보 | 혼자 정리하거나 관찰한 뒤 참여하고, 표현 전에 상황·확신·차례를 더 살피는 경향 | 사람을 싫어한다, 소통을 못한다, 소극적이라 나쁘다 |

E와 I 후보는 우열이 아니다. 두 방향은 에너지를 관리하고 표현을 시작하는 방식의 차이다.

## 4. SE-RE 구성개념 명세

### 4.1 식별 정보

- ID: `SE-RE`
- 권장 내부 명칭: `교류 활력`
- 고객 명칭: `함께할 때의 에너지`
- 이론적 대응 가설: social engagement / enthusiasm 계열
- 측정 단위: 성격 전체가 아니라 일상적인 대인 교류에서의 전형적 활력·관여 경향

### 4.2 한 문장 정의

`SE-RE 교류 활력`은 일상적인 대화나 공동 활동에서 사람들과 함께할수록 활력이 올라오고 그 상호작용에 관여하는 경향이다.

### 4.3 높은 방향에서 관찰 가능한 모습

- 대화가 이어지면서 기분이나 에너지가 살아나는 경우가 비교적 많다.
- 함께하는 활동에 들어가면 관찰만 하기보다 자연스럽게 관여하는 편이다.
- 익숙한 사람과의 교류뿐 아니라 부담이 크지 않은 새 교류에도 비교적 쉽게 들어간다.
- 사람들과 시간을 보낸 뒤 피로만 남기보다 활력을 얻는 경우가 많다.

### 4.4 낮은 방향에서 관찰 가능한 모습

- 대인 교류가 활력을 크게 올리지 않거나, 교류 뒤 혼자 회복할 시간이 필요한 경우가 많다.
- 새로운 자리에서는 바로 섞이기보다 먼저 분위기와 사람을 살핀다.
- 함께하는 활동을 싫어하지 않더라도 참여 강도와 시간을 조절하는 편이다.
- 가까운 관계를 중요하게 여기면서도 에너지는 혼자 있는 시간에서 더 잘 회복할 수 있다.

### 4.5 포함하는 내용

- `활력 변화`: 교류 중·후 에너지가 살아나는가, 소모되는가
- `교류 관여`: 대화나 공동 활동에 자연스럽게 빠져드는가
- `참여 접근`: 부담이 크지 않은 일상 교류에 들어가는 것이 자연스러운가
- `사회적 즐거움`: 사람과 함께하는 바로 그 상황에서 긍정 정서가 활성화되는가

### 4.6 포함하지 않는 내용

- 감정을 크게 드러내는 정도 자체
- 먼저 말하거나 진행을 이끄는 정도
- 낯선 사람에게 능숙하게 다가가는 기술
- 파티, 회식, 단체 모임을 좋아하는 취향
- 친구의 수나 연락 빈도
- 상대 마음을 읽고 배려하는 정도
- 거절당할까 걱정하는 정도
- 평소의 전반적인 행복감이나 활동량

### 4.7 인접 성향과의 경계

| 혼동 대상 | SE-RE와 구분하는 질문 |
|---|---|
| SE-AI 주도적 표현 | `에너지가 올라오는가`를 묻는가, `먼저 표현하는가`를 묻는가 |
| RO-EC 공감적 배려 | 함께 있을 때 활력을 얻는가, 상대 감정과 부담을 살피는가 |
| ER-WD 걱정·위축 | 원래 교류로 에너지가 소모되는가, 걱정이나 두려움 때문에 피하는가 |
| OE 탐색 성향 | 사람과의 상호작용에서 활력을 얻는가, 새로운 자극·생각 자체를 탐색하는가 |
| 전반적 긍정 정서 | 사람과 함께하는 특정 맥락의 활성화인가, 상황과 무관한 기분인가 |

### 4.8 허용 상황과 피할 상황

허용 상황:

- 익숙한 사람들과 이어지는 대화
- 처음 만난 소규모 자리
- 취미·과제·약속 등 함께하는 활동
- 일정 시간 교류한 뒤의 에너지 상태
- 관심 있는 주제로 대화가 길어지는 장면

피할 상황:

- 회식, 클럽, 파티처럼 특정 문화·취향에 치우친 장면
- 발표, 면접, 영업처럼 능력을 평가받는 장면
- 연애, 직장, 학교에만 해당하는 장면
- 갈등, 거절, 비난처럼 ER-WD의 불안 반응을 강하게 유발하는 장면
- 상대를 배려하거나 도와야 하는 도덕적 장면

## 5. SE-AI 구성개념 명세

### 5.1 식별 정보

- ID: `SE-AI`
- 권장 내부 명칭: `주도적 표현`
- 고객 명칭: `먼저 표현하기`
- 이론적 대응 가설: assertive initiation / expression 계열
- 측정 단위: 능력이나 성과가 아니라 필요한 대인 상황에서 먼저 표현하는 전형적 경향

### 5.2 한 문장 정의

`SE-AI 주도적 표현`은 공동 결정이나 조율이 필요한 상황에서 자신의 의견·요청·선택지 또는 진행 방법을 먼저 밖으로 꺼내는 경향이다.

### 5.3 높은 방향에서 관찰 가능한 모습

- 여럿이 결정을 미룰 때 선택지를 먼저 제안하는 편이다.
- 의견이 다를 때 침묵하기보다 자신의 생각을 말하는 편이다.
- 필요한 것이 있으면 상대가 알아차리기만 기다리지 않고 요청하는 편이다.
- 진행 방향이 불분명할 때 다음 방법을 제안하는 편이다.

### 5.4 낮은 방향에서 관찰 가능한 모습

- 다른 사람이 먼저 의견을 낸 뒤 자신의 생각을 더하는 편이다.
- 확신이나 발언할 차례가 생길 때까지 상황을 살핀다.
- 조율이 필요해도 먼저 방향을 정하기보다 제안된 선택지에 반응하는 편이다.
- 요청 전에 상대의 반응이나 관계 맥락을 더 오래 고려한다.

### 5.5 포함하는 내용

- `제안 시작`: 선택지·주제·다음 행동을 먼저 꺼냄
- `의견 표현`: 다른 생각을 필요한 때 밝힘
- `필요 표현`: 필요한 도움·조건·요청을 직접 말함
- `과정 제시`: 함께 움직여야 할 때 진행 방법을 제안함

### 5.6 포함하지 않는 내용

- 말하기, 발표, 토론, 설득을 잘하는 능력
- 실제로 사람을 이끌어 성과를 내는 리더십
- 다른 사람을 통제하거나 자신의 의견을 관철하는 지배성
- 공격성, 언성 높이기, 갈등을 감수하는 용기
- 자신감이나 자존감 전체
- 사람들과 함께할 때 얻는 에너지
- 공감, 배려, 존중, 예의
- 계획을 세우고 실행하는 과업 주도성

### 5.7 인접 성향과의 경계

| 혼동 대상 | SE-AI와 구분하는 질문 |
|---|---|
| SE-RE 교류 활력 | 먼저 표현하는가, 사람과 함께할 때 활력이 올라오는가 |
| RO-RN 존중·비강압 | 자신의 의견을 꺼내는가, 상대의 거절·선택을 존중하는가 |
| RO-EC 공감적 배려 | 말이나 요청을 먼저 하는가, 상대의 감정과 부담을 살피는가 |
| SM-EP 실행·지속 | 대인 장면에서 제안을 시작하는가, 혼자서도 과업을 시작·지속하는가 |
| ER-WD 걱정·위축 | 평소 표현 순서의 선호인가, 불안·부정적 결과 예상 때문에 억제되는가 |
| 리더십 능력 | 표현을 시작하는 경향인가, 실제로 사람을 조율해 성과를 내는 능력인가 |

### 5.8 허용 상황과 피할 상황

허용 상황:

- 여럿이 결정을 미루는 장면
- 가벼운 의견 차이가 생긴 장면
- 필요한 도움이나 조건을 말해야 하는 장면
- 모임이나 공동 활동의 다음 순서가 불분명한 장면
- 대화나 활동을 시작할 사람이 정해지지 않은 장면

피할 상황:

- 상사·교사·부모처럼 권력 차이가 응답을 결정하는 장면
- 생계, 성적, 안전이 걸린 중대한 갈등
- 전문 발표나 토론 대회처럼 능력 차이가 큰 장면
- 상대의 권리를 침해해야만 높은 방향이 되는 장면
- 과업 실행력이나 계획성을 함께 요구하는 복합 장면

## 6. 두 세부 성향의 조합

두 세부 성향의 차이를 없애는 평균 리포트는 사용하지 않는다.

| SE-RE 함께할 때의 에너지 | SE-AI 먼저 표현하기 | 설명 원칙 | 고객에게 가능한 설명 예시 |
|---|---|---|---|
| 높음 | 높음 | 교류에서 활력을 얻고 표현도 먼저 시작함 | `사람들과 함께할 때 활력이 살아나고, 필요한 의견도 비교적 먼저 꺼내는 편이에요.` |
| 높음 | 낮음 | 교류는 즐기지만 진행과 첫 표현은 다른 사람에게 맡김 | `함께하는 시간에서 에너지를 얻지만, 의견은 상황을 본 뒤 보태는 편일 수 있어요.` |
| 낮음 | 높음 | 교류 시간은 조절하지만 필요할 때 표현은 직접 시작함 | `혼자 정리하는 시간이 필요해도, 필요한 의견이나 요청은 먼저 분명히 꺼낼 수 있어요.` |
| 낮음 | 낮음 | 에너지를 아끼며 관찰·정리 후 표현함 | `사람이 많은 자리에서는 에너지를 조절하고, 충분히 살핀 뒤 의견을 말하는 편일 수 있어요.` |

네 조합 모두 정상적인 개인차다. `높음+높음`을 이상적인 유형처럼 시각화하거나 보상하지 않는다.

## 7. 결과 리포트 주장 계약

### 7.1 측정 근거가 확보되면 말할 수 있는 범위

- 사람과 함께할 때 활력이 올라오는 경향
- 교류 뒤 혼자 회복할 시간이 필요한 경향
- 의견이나 선택지를 먼저 꺼내는 경향
- 상황을 살핀 뒤 의견을 보태는 경향
- 두 세부 성향이 비슷하게 움직이는지, 서로 다른 방향을 보이는지

### 7.2 말하면 안 되는 주장

- `사교성이 좋다/나쁘다`
- `리더형이다`, `리더십이 부족하다`
- `사람을 좋아한다/싫어한다`
- `친구가 많다/적다`
- `관계를 잘한다/못한다`
- `말을 잘한다/못한다`
- `인기가 많다`, `매력적이다`
- `내향형이므로 사회생활이 어렵다`
- `외향형이므로 협업을 잘한다`
- SE 점수만으로 특정 직업·연애 방식·성공 가능성을 권하는 문구

### 7.3 문구 톤

- `~이다`보다 `~하는 편일 수 있어요`, `~하는 경향이 보여요`를 사용한다.
- 낮은 방향을 결핍이나 개선 대상으로 표현하지 않는다.
- 빠른 검사 결과에는 `현재 응답에서 보인 예비 방향`임을 표시한다.
- 불확실성이 큰 경우 설명을 줄이고 정밀 검사를 제안한다.

## 8. 문항 제작을 위한 상황 커버리지

M03에서 같은 장면만 반복해 신뢰도가 높아 보이는 오류를 피한다.

### 8.1 SE-RE 후보 버킷

| 버킷 | 관찰 목표 | 오염 위험 |
|---|---|---|
| 익숙한 사람과의 대화 | 교류가 이어질 때의 활력 변화 | 관계 친밀도 |
| 처음 만난 소규모 자리 | 부담이 크지 않은 교류 진입 | 사회불안, 대화 기술 |
| 공동 활동 | 함께할 때의 관여와 즐거움 | 활동 자체의 취향 |
| 교류 이후 | 회복 또는 활성화 방향 | 수면, 일정, 신체 피로 |
| 길어진 상호작용 | 지속 교류에서의 에너지 변화 | 주제 관심도 |

### 8.2 SE-AI 후보 버킷

| 버킷 | 관찰 목표 | 오염 위험 |
|---|---|---|
| 결정 지연 | 선택지를 먼저 제안함 | 계획성, 리더 역할 |
| 가벼운 의견 차이 | 다른 의견을 밖으로 표현함 | 갈등 불안, 공격성 |
| 필요·요청 | 필요한 것을 직접 말함 | 관계 친밀도, 예의 규범 |
| 방향 불명확 | 진행 방법을 먼저 제안함 | 과업 능력, 책임 역할 |
| 시작 공백 | 대화·활동의 첫 행동을 시작함 | SE-RE, 충동성 |

각 문항은 한 버킷, 한 상황, 한 반응만 가진다. `상황 라벨`은 조건만, `질문 문장`은 반응만 담는다.

아래 평가는 문항을 삭제한다는 최종 결정이 아니다. M03 후보 은행에 그대로 승계할 수 있는지 판단하는 1차 construct audit다.

- 그대로 운영 승인할 문항은 없다.
- `SERE-02`, `SERE-07`, `SEAI-01`, `SEAI-04`는 목표 반응이 비교적 분명해 새 사양의 출발점으로만 사용한다.
- 나머지도 폐기 전 후보로 보존하지만, 새 구성개념 정의에 맞춰 처음부터 경쟁시킨다.
- 문항이 영어 공개 원문에서 출발했더라도 한국어 표현과 뉴앙 구조의 타당성은 별도로 검증한다.

## 10. 채점·코드·빠른 검사 계약

### 10.1 문항에서 코드까지

```text
SE-RE 문항들 → SE-RE 연속 점수 + 오차
SE-AI 문항들 → SE-AI 연속 점수 + 오차
두 facet의 구조·가중치 검증 → SE 상위 점수 + 오차
경계·분기 규칙 검증 → E 또는 I, 혹은 확정 보류
검증된 점수 범위 → 허용된 리포트 문구
```

- 문항 하나가 E/I를 직접 판정하지 않는다.
- 두 facet 중 하나의 증거가 부족하면 SE 전체를 확정하지 않는다.
- 두 facet이 크게 갈리면 단순 평균만으로 강한 E/I 설명을 하지 않는다.
- 현재의 동일 가중 평균과 50점 이분법은 M07·M08 검증 전 가설이다.

### 10.2 빠른 코어의 제한

빠른 코어는 현재 facet당 2문항뿐이다. 짧은 성격 척도 연구에서도 2문항 facet은 개인의 세부 성향을 확정하기보다 큰 표본의 facet 수준 분석에서 제한적으로 유용하며, 1문항 facet은 domain 평가에만 사용하도록 경고한다.

따라서 뉴앙의 독립 검증 전에는 다음을 적용한다.

- 빠른 결과: SE 상위 방향의 `예비 신호`만 제공
- 빠른 facet: 내부 계산·정밀 검사 추천에 사용할 수 있으나 고객에게 확정 문구로 노출하지 않음
- 정밀 결과: 충분한 문항과 타당성 근거가 확보된 facet만 설명
- 기존 `minValidResponses=1`: 운영 release 전 반드시 상향 또는 결과 보류 규칙으로 교체

## 12. 사용자 승인 기록

2026-07-17 KST 사용자 검토를 거쳐 다음 일곱 항목을 제품 설계로 확정했다. 단, 실제 점수 경계와 문항은 파일럿 검증 전까지 운영에 사용하지 않는다.

- [x] 상위 축 `사람 사이 에너지`를 유지한다.
- [x] 첫 코드 글자는 E/I, 고객 자리 순서는 `SE → OE → RO → SM → ER`로 관리한다. 현재 작업 범례는 `E/I · R/N · G/A · K/M · C/Q`다.
- [x] SE-RE 내부 명칭을 `교류 활력`, 고객 명칭을 `함께할 때의 에너지`로 변경한다.

- [x] 두 facet이 다르면 `두 반응이 다르게 나타나요`라고 보여주고 평균으로 차이를 숨기지 않는다.
- [x] 빠른 코어에서는 검증 전 facet을 확정적으로 설명하지 않는다.

## 13. 승인 후 다음 단계

SE 승인 뒤 RO와 ER 구성개념 검토도 사용자 승인까지 완료했다. 다음은 `M01-SM 일상을 꾸리는 방식`을 진행한다. 모든 M01 영역을 닫은 뒤 아래의 `M02 문항 사양과 UI 문법`으로 이동한다.

1. SE-RE의 상황 버킷별 `상황 라벨 + 단일 반응` 문법을 만든다.
2. SE-AI에 같은 작업을 수행한다.
3. 라벨·질문 중복, 인접 성향 오염, 역할·권력 의존을 자동 점검할 규칙을 만든다.
4. 사용자가 사양을 승인하면 M03에서 SE-RE 후보 문항부터 한 세부 성향씩 작성한다.
