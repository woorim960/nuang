# SMRL-C11-r3 제한 M04 내부 판정

상태: `PASS_TO_COGNITIVE_NOT_EXTERNAL_VALIDATION`  
protocol: `m04-core-expert-kit.v0.2-smrl-r3`

## 판정

`SMRL-C11-r3`은 현재 문구로 M05 사용자 이해 검토에 이동한다.

이 판정은 운영 문항 승인이나 타당도 확보가 아니다. 외부 사용자가 상황과 질문을 의도대로 이해하는지 확인할 수 있을 정도로 내부 target mapping이 안정됐다는 뜻이다.

## 근거

- blind 첫 구성개념 `SM_RL`: 6/6
- blind `HIGH` 방향 일치: 6/6
- target 관련성 median: 4/4
- 방향 적합성 median: 4/4
- 명확성 median: 4/4
- 단일 반응성·보편성·6개월 빈도 적합성 median: 각각 3/4
- 인접 성향 분리 median: 3/4
- Stage 2 권고: `KEEP` 4, `COPY_REVISE` 2
- fatal risk: 0

## 보존할 반대 의견

1. 4/6이 두 번째 구성개념으로 `SM_EP`를 골라 일반적인 실행·마무리와의 seam이 남아 있다.
2. 정한 때에 끝낼 수 있는지는 일정 통제권, 과제 난도, 건강·돌봄과 같은 수행 여건에 영향을 받을 수 있다.
3. 함께 맡은 일의 경험이 적으면 최근 6개월 빈도가 성향보다 노출 기회를 반영할 수 있다.
4. 정한 때에 끝내는 답은 책임감 있는 사람처럼 보이기 쉬워 사회적 바람직성이 남는다.
5. `정한 때에 맞춰`보다 `정한 때까지`가 더 분명하다는 소수 문구 의견을 M05에서 확인한다.

## M05 필수 확인

- 일반적으로 일을 잘 마무리하는 것과 다른 사람과 정한 역할을 이행하는 것을 구분해 답했는가?
- 본인이 시간을 조정할 수 없었던 상황이나 건강·돌봄·과제 난도가 답을 바꿨는가?
- 최근 6개월에 함께 맡은 일이 거의 없다면 `판단하기 어려워요`가 필요했는가?
- 어느 답이 더 책임감 있고 좋은 사람처럼 보였는가? 결과가 공개되면 답을 바꾸고 싶은가?
- `정한 때에 맞춰`를 합의된 시점 이행으로 이해했는가? `정한 때까지`가 더 즉각적인가?

M05에서 반복 S2 또는 미해결 S3 문제가 나오면 `COPY_REVISE_RETEST`, `HOLD_FOR_SUBGROUP`, `CONSTRUCT_REWRITE`, `EXCLUDE` 중 하나로 되돌린다.
