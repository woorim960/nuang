## mapping 순서

1. 상황 라벨과 질문을 하나의 의미 단위로 읽는다.
2. 잘하는 능력이 아니라 반복해서 나타나는 반응을 본다.
3. 가장 직접적인 구성개념 하나를 첫 선택으로 고른다.
4. 강한 겹침이 있을 때만 두 번째를 고르고, 아니면 `NONE`으로 둔다.
5. 능력·환경·문장 형식이 답을 지배하면 `METHOD`를 우선 검토한다.

공개 코드 글자나 결과 이름은 Stage 1 판단에 사용하지 않는다.
