## E. 사전 판정식

유효 reviewer 수 `n`에 대해:

- target first mapping: `ceil(n × 0.75)` 이상
- target 관련성 중앙값: 3 이상
- 명확성 중앙값: 3 이상
- 단일 반응성 중앙값: 3 이상
- 해결되지 않은 fatal risk: 0
- 동일 비target first mapping: `ceil(n × 0.30)` 이상이면 seam flag

| 제안 결정              | 기준                                                        |
| ---------------------- | ----------------------------------------------------------- |
| `PASS_TO_COGNITIVE`    | 통과 기준 충족, 중대한 역할·접근 위험 없음                  |
| `COPY_REVISE`          | target은 대체로 맞지만 문구·척도·방향 오해 반복             |
| `CONSTRUCT_REWRITE`    | target 일치 50~74% 또는 동일 인접 facet 반복                |
| `HOLD_FOR_ACCESS`      | 접근·경험·생활 조건이 판단을 지배                           |
| `HOLD_FOR_RISK_REVIEW` | 한 명 이상이 구체적인 fatal risk 근거를 제시                |
| `EXCLUDE_OR_REBUILD`   | target 일치 50% 미만, 다른 성향·능력 중심, 미해결 치명 위험 |
