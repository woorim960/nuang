# Stage 1 blind 구성개념 코드북

문서 버전: `m04-oe-expert-kit.v0.1`  
사용 범위: Stage 1 첫 번째·두 번째 구성개념 mapping  
원칙: 각 정의는 중립적이다. 어느 방향도 더 건강하거나 우수하다는 뜻이 아니다.

## 선택 가능한 구성개념

| 입력 코드 | 화면 명칭           | 판단 기준                                                         | 포함하지 않는 것                                  |
| --------- | ------------------- | ----------------------------------------------------------------- | ------------------------------------------------- |
| `OE_AE`   | 미적 경험           | 실제 장면·소리·공간의 분위기와 인상에 관심이 머무는 경향          | 시력·청력, 예술 지식, 아름다움을 잘 판별하는 능력 |
| `OE_CI`   | 상상 확장           | 주어진 내용 바깥의 장면·이야기·가능성을 새로 이어서 떠올리는 경향 | 기억력, 시각 심상 능력, 현실 판단 능력            |
| `OE_IE`   | 지적 탐구           | 개념·이유·배경·서로 다른 설명을 필요한 범위보다 더 알아보는 경향  | 지능, 학력, 검색 기술, 많이 아는 정도             |
| `ER_IR`   | 감정 동요           | 좌절·변화에서 불편한 감정이 빠르게 커지거나 오래 이어지는 경향    | 감정이 풍부함 전체, 임상 진단                     |
| `ER_WD`   | 걱정·주저           | 불확실성·평가에서 잘못될 가능성이 반복되어 결정을 주저하는 경향   | 신중함 전체, 위험을 정확히 계산하는 능력          |
| `SM`      | 실행·지속           | 정한 행동을 시작하고, 중단 후 복귀하며, 지속·마무리하는 경향      | 생산성·성공·시간 여유, 일의 난이도                |
| `RO`      | 선택·존엄 존중      | 관계에서 상대의 마음과 선택 여지를 먼저 고려하는 경향             | 무조건 동의함, 착함, 갈등을 피함                  |
| `METHOD`  | 능력·접근·방법 효과 | 감각·심상·지능·기억·읽기·검색·문장 형식이 답을 주로 결정함        | 성향 구성개념으로 억지 분류하지 않음              |
| `NONE`    | 명확히 맞지 않음    | 위 정의 중 하나로 안정적으로 분류하기 어려움                      | 가장 비슷한 답을 억지로 선택하지 않음             |

## mapping 순서

1. 상황 라벨과 질문을 한 의미 단위로 읽는다.
2. `무엇을 잘하는가`가 아니라 `어떤 반응이 자주 나타나는가`를 본다.
3. 첫 번째 선택에는 문항이 가장 직접적으로 묻는 한 가지를 고른다.
4. 두 번째 선택은 실제로 강하게 겹칠 때만 고르고, 없으면 `NONE`으로 둔다.
5. 능력·노출·접근·매체가 없으면 답할 수 없는 문항은 `METHOD` 가능성을 우선 검토한다.

## 자주 생기는 혼동

- 분위기를 알아차리는 반응은 `OE_AE`, 앞뒤 이야기를 새로 만드는 반응은 `OE_CI`다.
- 이유나 원리를 더 확인하는 반응은 `OE_IE`, 틀릴까 봐 반복 확인하는 반응은 `ER_WD`일 수 있다.
- 해야 해서 정보를 찾는 것만으로는 `OE_IE`가 아니다. 필요한 범위를 넘어 더 알아보는 관심이 있어야 한다.
- 상상을 쉽게 할 수 있는 능력과 실제로 이야기를 확장하려는 경향은 다르다.
- 질문이 쉬워 보여도 특정 감각·검색·독서 환경에 크게 의존하면 `METHOD` flag가 필요하다.
