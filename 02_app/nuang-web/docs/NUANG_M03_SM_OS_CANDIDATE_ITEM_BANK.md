## 0. 이번 단계의 결론

SM-OS는 깨끗함·꼼꼼함·완벽주의나 MBTI J/P를 재는 검사가 아니다.

```text
물건·시간·해야 할 일·반복 절차
→ 다시 찾거나 이어가기 쉬운 위치·순서·때를 정하고
→ 비슷한 상황에서 그 구조를 반복해 사용하는 경향
```

첫 후보 은행은 6개 상황 버킷, 총 12문항이다.

| 구분                   |  수 |
| ---------------------- | --: |
| 높은 SM-OS 방향 `HIGH` |   6 |
| 낮은 SM-OS 방향 `LOW`  |   6 |
| 주력 인지 인터뷰 후보  |   8 |
| 경계 확인용 탐색 후보  |   4 |

LOW는 지저분함이나 생활 능력 부족이 아니다. 위치·순서·때를 미리 고정하는 정도가 낮고 그때 쓰기 편한 곳이나 필요한 방식으로 처리하는 경향의 증거 후보다. 이 사실만으로 변화 적응력이나 창의성을 추론하지 않는다.

## 1. 측정 목표와 금지 해석

### 측정하려는 것

- 반복해서 쓰는 물건·자료의 위치를 정하는 경향
- 여러 할 일을 순서·묶음으로 구조화하는 경향
- 반복 업무에 재사용할 순서를 만드는 경향
- 가까운 미래의 할 시점을 대략 정하는 경향
- 다시 쓸 자료를 찾을 수 있게 분류하는 경향
- 여러 단계의 일을 일정한 순서로 확인하는 경향

### 측정하지 않는 것

- 청결·위생·인테리어·미적 취향
- 완벽해야 안심되는 강박적 요구
- 계획의 품질, 시간관리 능력, 실제 성과
- 특정 캘린더·메모·생산성 앱 사용 경험
- 직장·학교가 강제한 절차 준수
- 갑작스러운 변화에 대한 감정 반응
- 선택지를 열어두는 MBTI P 선호
- 시작하고 계속하는 SM-EP

## 2. 공통 응답 계약

SM-EP와 동일한 `최근 6개월 조건부 빈도 1~5 + 이 상황은 답하기 어려워요`를 사용한다.

- 자기만의 물건·자료·일정이 거의 없는 환경은 경험 부족으로 기록한다.
- 다른 사람이 정한 위치·절차를 따를 수밖에 없었던 경우는 자발적 SM-OS로 단정하지 않는다.
- 종이·디지털 등 특정 도구를 쓰는지는 점수 방향이 아니다.
- 같은 상황의 HIGH/LOW 후보는 한 form에 함께 제시하지 않는다.

## 6. HIGH·LOW와 K/M의 비우열 계약

| 방향          | 후보가 직접 말할 수 있는 것                             | 금지되는 단정                               |
| ------------- | ------------------------------------------------------- | ------------------------------------------- |
| HIGH → K 증거 | 위치·순서·때를 정해 반복 사용하는 경향                  | 깔끔함, 정확함, 계획을 잘 세움, J형, 유능함 |
| LOW → M 증거  | 현재 쓰임·단서·상황에 따라 위치·순서·때가 달라지는 경향 | 지저분함, 산만함, P형, 변화 적응력이 좋음   |

양쪽에 임의의 장점을 붙여 대칭처럼 보이게 하지 않는다. 문항이 직접 관찰한 구조 사용 범위만 설명한다.

공통 확인 질문:

1. 자신이 위치·순서·시점을 정할 권한이 있는 장면이었는가?
2. 특정 앱·메모 방식·직장 규칙 때문에 응답했는가?
3. 정리의 기준이 청결·미관·불안 해소였는가, 다시 쓰기 위한 구조였는가?
4. HIGH는 유능하고 LOW는 유연해 보인다고 느꼈는가?
5. C07·C08을 계획 시점이 아니라 미루기 문항으로 이해했는가?
