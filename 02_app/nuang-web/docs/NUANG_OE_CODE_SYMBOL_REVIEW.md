## 0. 권장안

OE 두 번째 자리의 권장 문자쌍은 다음과 같다.

```text
R / N
```

고객에게 보이는 필수 한국어:

```text
R  익숙하고 구체적인 것에 관심
N  새로운 관점과 가능성에 관심
```

N은 MBTI N과 높은 방향이 유사하지만 같은 검사 결과가 아니다. R은 현실 감각이나 실용 능력이 아니라, 새롭고 추상적이거나 미적인 탐색에 비해 익숙하고 구체적인 대상에 관심이 더 머무는 방향을 나타낸다.

고객 화면에서는 `R = Real-world`, `N = New-perspective` 같은 영어 확장어를 기본 노출하지 않는다. 영어는 내부 기억 단서일 뿐이고, 실제 의미 계약은 위 한국어 문장과 세 facet 점수다.

## 1. 문자쌍이 요약하는 측정 범위

OE는 사용자 결정에 따라 세 facet으로 구성한다.

| ID | 내부 명칭 | 고객 명칭 | 측정 범위 |
|---|---|---|---|
| OE-AE | 미적 경험 | 분위기와 아름다움 느끼기 | 음악·장면·작품·환경의 분위기와 아름다움에 관심이 가는 정도 |
| OE-CI | 상상 확장 | 장면과 이야기 떠올리기 | 실제로 없는 장면·이야기·가능성을 머릿속에서 펼치는 것을 즐기는 정도 |
| OE-IE | 지적 탐구 | 원리와 관점 알아보기 | 새로운 개념·원리·대안 설명을 알아보는 것을 즐기는 정도 |

R/N은 세 점수를 지워버리는 유형명이 아니라 상위 OE 연속 점수를 기억하기 위한 두 번째 글자다. 결과 화면에는 R/N과 함께 세 facet의 방향과 선명도를 제공한다.

## 2. 문자 선정 기준

후보는 아래 조건으로 검토했다.

1. N이 MBTI N과 반대 방향으로 읽히지 않는가
2. 미적 경험·상상 확장·지적 탐구를 모두 포함할 수 있는가
3. 낮은 방향에 현실성·실용 능력·상식 같은 측정하지 않은 장점을 넣지 않는가
4. 높은 방향에 지능·창의력·통찰력 같은 측정하지 않은 능력을 넣지 않는가
5. 현재 범례 `E/I · _/_ · G/A · K/M · C/Q`의 다른 글자와 충돌하지 않는가
6. MBTI의 익숙한 글자를 다른 의미로 바꿔 쓰지 않는가
7. 쉬운 한국어를 함께 봤을 때 방향을 기억하고 설명할 수 있는가
8. 어느 한쪽이 더 좋은 등급처럼 느껴지지 않는가

## 3. 후보 비교

| 후보 | 기억 연결 | 장점 | 핵심 문제 | 판정 |
|---|---|---|---|---|
| `R/N` | 익숙하고 구체적 / 새로운 관점과 가능성 | N의 익숙한 방향을 유지하고 전체 코드의 다른 글자와 겹치지 않음 | R을 현실 능력으로 오해할 수 있어 한국어 설명이 항상 필요 | 권장 |
| `S/N` | Sensing / Intuition | MBTI 사용자에게 즉시 익숙함 | 뉴앙 OE는 정보 수집 선호 전체가 아니며 미적 경험·상상·지적 탐구를 측정하므로 같은 검사 오해가 큼 | 탈락 |
| `F/N` | Familiar / New | 익숙한 것과 새로운 것의 대비가 의미상 단순함 | F가 MBTI Feeling으로 강하게 학습돼 있고 관계 자리와 혼동될 수 있음 | 보류 대안 |
| `C/N` | Concrete / New | 구체적이라는 뜻이 직접적임 | C는 뉴앙 다섯 번째 Calm과 충돌하고 MBTI S/N을 글자만 바꾼 것처럼 보임 | 탈락 |
| `P/N` | Practical / New | 일상 언어로 대비하기 쉬움 | 실용 능력·효율을 측정하지 않으며 P가 MBTI Perceiving과 충돌 | 탈락 |
| `R/O` | 익숙한 쪽 / Open | R/N의 현실 능력 오해를 일부 피할 수 있음 | N의 방향 친숙성을 버리고 O가 개방성의 우열 등급처럼 읽힘 | 탈락 |

R/N은 설명 없이도 완벽한 문자가 아니다. 다만 다른 후보보다 구성개념 왜곡과 기존 문자 충돌을 동시에 적게 만든다. 출시 전 이해도 검증에서 R의 현실 능력 오해가 충분히 낮아지지 않으면 `F/N`을 사전 고정된 대안으로 다시 비교한다.

## 4. R 방향의 정확한 의미

### 4.1 고객 의미

`R  익숙하고 구체적인 것에 관심`

### 4.2 정의

새롭고 추상적이거나 미적인 탐색보다 익숙하고 구체적으로 접할 수 있는 대상에 관심이 더 머무는 경향이다.

R은 세 facet에서 아래와 같이 해석한다.

- 분위기와 아름다움의 인상을 일부러 오래 탐색하는 빈도가 상대적으로 낮을 수 있음
- 실제로 없는 장면과 이야기를 머릿속에서 펼치는 빈도가 상대적으로 낮을 수 있음
- 당장 필요하지 않은 개념과 원리를 깊게 탐색하는 빈도가 상대적으로 낮을 수 있음

### 4.3 R이 뜻하지 않는 것

- 현실 판단이 정확함
- 실용 능력이나 상식이 뛰어남
- 사실·세부를 더 정확하게 기억함
- 정보를 간단히 정리하는 능력이 좋음
- 변화에 안정적으로 대응함
- 새로운 것을 싫어하거나 폐쇄적임
- MBTI S와 같은 결과임

`현실형`, `실용형`, `현실적인 사람`은 고객 문구와 캐릭터 이름으로 사용하지 않는다.

## 5. N 방향의 정확한 의미

### 5.1 고객 의미

`N  새로운 관점과 가능성에 관심`

### 5.2 정의

분위기·아름다움·상상 경험과 새로운 개념·원리·관점·가능성이 자연스럽게 관심을 끌고, 이를 더 탐색하는 경향이다.

### 5.3 N이 뜻하지 않는 것

- 직관이 정확함
- 패턴이나 가능성을 남들보다 잘 발견함
- 지능·창의력·통찰력이 높음
- 미래를 잘 예측함
- 예술적 재능이 뛰어남
- 현실성이 부족함
- MBTI N과 같은 결과임

`직관형`, `창의형`, `아이디어 천재`, `몽상가`는 고객 문구와 캐릭터 이름으로 사용하지 않는다.

## 6. N은 왜 유지하고 S는 왜 사용하지 않는가

MBTI 공식 설명에서 Sensing은 구체적 사실·세부·현재 경험을 받아들이는 선호이고, Intuition은 패턴·의미·가능성과 큰 그림에 주의를 두는 선호다.

뉴앙 N은 `새로운 관점·가능성·연결·추상적 원리`에 관심이 간다는 점에서 방향이 유사하다. 따라서 N을 반대 방향으로 사용하지 않는다.

그러나 뉴앙은 아래까지 별도로 측정한다.

- 분위기와 아름다움에 대한 관심
- 머릿속 장면과 이야기의 상상
- 개념과 원리의 탐색 관심

이는 MBTI의 정보 지각 선호와 동일하지 않다. S/N을 그대로 사용하면 익숙함은 높지만 `뉴앙 N = MBTI N`, `뉴앙 S = MBTI S`라는 잘못된 변환을 강화한다. N의 방향 친숙성만 활용하고 반대 글자는 R로 구분한다.

## 7. 세 facet과 R/N의 관계

문자 하나로 세 점수를 덮지 않는다.

| 상태 | 두 번째 글자 | 고객 설명 | 화면 처리 |
|---|---|---|---|
| 세 facet 모두 N 방향 | N | 새로운 관점과 가능성에 관심이 가는 모습이 비교적 선명해요 | 세 facet 상세 제공 |
| 세 facet 모두 R 방향 | R | 익숙하고 구체적인 대상에 관심이 머무는 모습이 비교적 선명해요 | 세 facet 상세 제공 |
| facet 방향이 갈림 | 상위 연속 점수가 가까운 쪽 | 세 가지 반응이 다르게 나타나요 | 세 점수를 따로 설명하고 추가 문항을 강제하지 않음 |
| 중심점 부근·신뢰구간 교차 | 점추정치가 가까운 쪽 | R과 N이 비슷해요 | 신뢰도가 낮을 때만 적응형 문항 권장 |
| 응답 부족·품질 실패 | 미발급 | 결과를 만들기 위한 응답이 더 필요해요 | 추가 응답 또는 재검사 |

예를 들어 미적 경험은 N, 상상 확장과 지적 탐구는 R에 가깝다면 `N형` 또는 `R형`이라고 끝내지 않는다.

```text
세 가지 반응이 다르게 나타나요
분위기와 아름다움에는 관심이 자주 가지만,
장면을 상상하거나 원리를 더 알아보는 일은 상대적으로 적어요.
```

## 8. 고객 화면 문구

### 8.1 첫 학습 화면

```text
생각과 탐색

R  익숙하고 구체적인 것에 관심
   익숙하고 직접 접할 수 있는 대상에 관심이 더 오래 머무는 편

N  새로운 관점과 가능성에 관심
   분위기·상상·새로운 원리와 관점에 관심이 더 오래 머무는 편
```

두 설명을 같은 높이와 같은 시각적 무게로 보여준다. R에는 도구·계산기·정답, N에는 전구·천재·예술가처럼 능력이나 우열을 암시하는 이미지를 사용하지 않는다.

### 8.2 결과 카드

R이 선명한 경우:

```text
R · 익숙하고 구체적인 것에 관심
새롭고 추상적인 탐색보다
익숙하고 직접 접할 수 있는 대상에 관심이 더 머무는 편이에요.
```

N이 선명한 경우:

```text
N · 새로운 관점과 가능성에 관심
분위기·상상·새로운 원리와 관점이 관심을 끌고,
이를 더 알아보는 일을 즐기는 편이에요.
```

### 8.3 사용하지 않는 문구

- R형 / N형
- 현실형 / 직관형
- 실용형 / 창의형
- 감각형 / 상상형
- 현실적인 판단 / 뛰어난 통찰
- 검증된 것을 믿음 / 미래를 내다봄
- MBTI의 S와 같음 / MBTI의 N과 같음

## 9. 코드 조립

R/N 승인에 따라 작업 범례는 다음과 같다.

```text
E/I · R/N · G/A · K/M · C/Q
```

예시:

```text
E N A K Q → ENAKQ
I R G M C → IRGMC
```

예시는 자리와 발음을 확인하기 위한 가상 조합이다. 전체 의미·문항·채점이 승인되기 전에는 운영 코드, 공유 URL, 프로필 이름에 사용하지 않는다.

## 10. 사용자 이해·회상 검증

### 10.1 10초 이해

R/N과 필수 한국어를 10초 동안 보여준 뒤 묻는다.

- R과 N을 각각 어떤 관심 방향으로 이해했는가
- R을 현실 판단·실용 능력으로 해석했는가
- N을 지능·창의력·직관 정확성으로 해석했는가
- 어느 방향이 더 좋은 등급처럼 느껴지는가

### 10.2 지연 회상

10분 뒤 가상 코드의 두 번째 글자와 의미를 다시 말하게 한다.

- N의 방향을 MBTI N과 반대로 기억하지 않는가
- R을 S 또는 현실 능력으로 바꾸어 설명하지 않는가
- 필수 한국어 없이도 핵심 관심 방향을 회상하는가

### 10.3 사례 분류

세 facet이 함께 움직이는 사례와 갈리는 사례를 섞어 보여준다.

- 분위기·상상·원리 탐색이 모두 자주 나타나는 사람
- 세 탐색이 일상 관심에서 차지하는 비중이 모두 작은 사람
- 아름다움에는 관심이 많지만 상상과 원리 탐색은 적은 사람
- 상상은 자주 하지만 미적 경험과 지적 탐구는 적은 사람

갈리는 사례를 억지로 R 또는 N 하나로만 설명하면 결과 카드와 세부 성향 교육을 보완한다.

### 10.4 사전 결정 규칙

- R/N의 방향 회상과 설명 정확도가 수용 가능해야 한다.
- R을 현실성·실용 능력으로 해석하는 비율이 사전에 정한 한도 아래여야 한다.
- N을 지능·창의력 우월로 해석하는 비율이 사전에 정한 한도 아래여야 한다.
- `뉴앙 R/N = MBTI S/N`이라고 변환하는 오해가 낮아야 한다.
- 한쪽이 더 우월한 등급처럼 느껴지지 않아야 한다.
- 설명을 고쳐도 R 오해가 충분히 줄지 않으면 사전에 정한 대안 `F/N`을 같은 과제로 비교한다.

정량 기준은 UX 리서치 표본과 과제를 설계할 때 결과를 보기 전에 고정한다.

## 11. 사용자 승인 기록

2026-07-17 KST 사용자 검토를 거쳐 다음 다섯 항목을 제품 설계로 승인했다.

1. OE 두 번째 문자쌍을 `R/N`으로 정한다.
2. 고객 필수 문구를 `R = 익숙하고 구체적인 것에 관심`, `N = 새로운 관점과 가능성에 관심`으로 정한다.
3. 고객 화면에는 `Real-world / New-perspective` 영어 확장어를 기본 노출하지 않는다.
4. R에서 현실성·실용 능력을, N에서 지능·창의력·통찰력을 추론하지 않는다.
5. 이해도 검증에서 R의 현실 능력 오해가 해결되지 않으면 사전 대안 `F/N`을 비교한다.

이 승인만으로 운영 코드를 바꾸지 않는다. 전체 코드 정합성 감사와 문항·채점·DB versioning 설계가 끝난 뒤 migration한다.
