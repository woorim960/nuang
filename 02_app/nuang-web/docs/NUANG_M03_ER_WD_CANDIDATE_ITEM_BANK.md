## 0. 이번 단계의 결론

ER-WD는 신중함·준비성·자신감·용기를 평가하지 않는다.

```text
결과가 정해지지 않음·답이 늦음·평가 가능성·확신하기 어려운 선택
→ 잘못될 가능성이 반복해서 떠오르고
→ 그 걱정이 시작·표현·결정을 다시 망설이게 하는 경향
```

첫 후보 은행은 6개 상황 버킷, 총 12문항으로 구성했다.

| 구분                   |  수 |
| ---------------------- | --: |
| 높은 ER-WD 방향 `HIGH` |   6 |
| 낮은 ER-WD 방향 `LOW`  |   6 |
| 주력 인지 인터뷰 후보  |   8 |
| 경계 확인용 탐색 후보  |   4 |

`HIGH`는 Q의 증거 후보, `LOW`는 C의 증거 후보다. HIGH가 위험을 더 정확히 예측한다는 뜻이 아니고, LOW가 용감하거나 낙관적이라는 뜻도 아니다.

## 1. 측정 목표와 금지 해석

### 측정하려는 것

- 불확실한 결과를 기다릴 때 잘못될 가능성이 반복되는 정도
- 상대의 답이 늦을 때 좋지 않은 이유를 되풀이해 추측하는 정도
- 평가 가능성 앞에서 해야 할 행동을 다시 망설이는 정도
- 확신하기 어려운 선택에서 같은 걱정으로 결정을 늦추는 정도
- 작은 실패 뒤 다시 잘되지 않을 장면이 반복되는 정도
- 감당 가능한 일이 조금 늘었을 때 걱정이 이어지는 정도

### 측정하지 않는 것

- 위험을 정확히 분석하는 능력이나 신중한 계획
- 일반적 자신감·용기·낙관성
- 의견을 먼저 말하는 SE-AI
- 과업을 시작·지속하는 SM-EP
- 여러 가능성을 즐겨 탐색하는 OE
- 사회불안·범불안·우울 장애 진단
- 절망·무가치감·흥미 저하·수면·식욕 같은 우울 선별 내용
- 자해·자살·공황·외상·강박 위험 선별

## 2. 공통 응답·안전 계약

- 최근 6개월의 평소 모습을 기준으로 한다.
- 실제 손실 가능성이 매우 크거나 안전이 위협받는 상황은 떠올리지 않는다.
- 걱정 때문에 같은 생각이 반복되거나 행동이 늦어진 것과, 필요한 정보를 계획적으로 확인한 것을 구분한다.
- 최근 2주의 예외적인 스트레스는 성향 점수와 별도 상태 정보로만 다룬다.
- 문항이 우울·불안 진단이나 위험 선별을 한다고 안내하지 않는다.
- 응답하지 않거나 중단해도 불이익이 없으며 `이 상황은 답하기 어려워요`를 정식 상태로 둔다.

## 6. HIGH·LOW와 C/Q의 비진단 계약

| 방향          | 후보가 직접 말할 수 있는 것                                                | 금지되는 단정                                    |
| ------------- | -------------------------------------------------------------------------- | ------------------------------------------------ |
| HIGH → Q 증거 | 불확실성·평가·작은 실패 앞에서 잘못될 생각이 반복되거나 행동을 망설이게 함 | 불안 장애, 소심함, 위험 판단이 좋음, 의지가 약함 |
| LOW → C 증거  | 같은 상황에서 걱정이 비교적 짧게 이어지고 행동을 멈추게 하는 일이 적음     | 용감함, 자신감이 높음, 낙관적임, 준비를 잘함     |

LOW 응답자가 실제 위험을 무시한다거나 HIGH 응답자가 위험을 잘 발견한다고 설명하지 않는다.
